Beskrivning av HTTP, TCP och IP

HTTP - Hypertext Transfer Protocol

HTTP används av datorerna för att ta kontakt med varandra, be om och skicka information. Med HTTP upprättar datorerna kontakt med varandra och håller sedan den kontakten tills de är klara.

TCP - Transmission Control Protocol

TCP delar upp information som ska skickas över nätet i mindre delar och håller koll på vilken ordning delarna ska vara i. När en dator tar emot information från webben sätter sen TCP ihop informationen igen. Om en del saknas behöver datorn då bara be om att en liten del ska skickas om.

IP - Internet Protocol

IP används av datorer för att hitta varandra över nätet. Protokollet tolkar webbadresser som människor skriver in och letar fram IP adressen till serven den webbadressen går till. IP adressen kan sedan användas av datorn för att hitta servern.
